# <u>Freestyle for USA</u>
This my ongoing project for connecting freestylers in the US.  

## <u>Technologies</u>
Project is created with:
* Node 17.4.0
* Mongoose 6.2.0
* Express 4.17.2
* Passport 0.5.2
* Axios 0.26.1
* Socket.IO 4.4.1
